================================================================================
PETUNJUK PENGGANTIAN DATA PLACEHOLDER - HALAMAN PENGEMBANG
================================================================================

Halaman Tim Pengembang pada sistem ini menggunakan data placeholder yang perlu
diganti dengan informasi sebenarnya sebelum pengumpulan tugas akhir.

--------------------------------------------------------------------------------
DATA YANG PERLU DIGANTI
--------------------------------------------------------------------------------

1. HALAMAN OVERVIEW (pengembang.html)
   Untuk setiap kartu pengembang, ganti:
   - Nama Pengembang 1, 2, 3, 4  →  Nama lengkap anggota tim
   - Project Lead, Backend Developer, dll  →  Peran sebenarnya dalam tim
   - Deskripsi singkat  →  Ringkasan kontribusi masing-masing anggota

2. HALAMAN DETAIL (pengembang_detail.html)
   Ganti semua placeholder dalam kurung siku [...] dengan data asli:
   - [Nama Pengembang]  →  Nama lengkap mahasiswa
   - [Nomor Induk Mahasiswa]  →  NIM/NPM yang valid
   - [Nama Program Studi]  →  Program studi mahasiswa
   - [Nama Fakultas]  →  Fakultas yang sesuai
   - [Nama Universitas]  →  Institusi pendidikan
   - [Tahun Angkatan]  →  Tahun masuk kuliah
   - [Jabatan dalam Tim]  →  Peran dalam proyek
   - Deskripsi kontribusi  →  Penjelasan tugas dan hasil kerja
   - Informasi kontak (email, GitHub, LinkedIn)  →  Data valid atau hapus

3. BADGE KEAHLIAN
   Sesuaikan daftar keahlian teknis dengan kemampuan sebenarnya dari
   masing-masing anggota tim.

--------------------------------------------------------------------------------
CATATAN PENTING
--------------------------------------------------------------------------------

- Pastikan semua data sudah akurat sebelum demo atau pengumpulan.
- Alert placeholder di bagian atas halaman dapat dihapus setelah data diganti.
- Foto profil dapat ditambahkan dengan mengganti ikon Bootstrap dengan tag <img>.
